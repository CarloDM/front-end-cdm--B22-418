CSS Grid

CSS Grid è un sistema di layout a due dimensioni: possiamo controllare righe e colonne insieme.

1) Cos'è CSS Grid

CSS Grid è un sistema di layout a due dimensioni: ci permette di controllare contemporaneamente righe e colonne.
Possiamo decidere la struttura del layout come fosse una tabella, ma con molta flessibilità.

Usiamo Grid quando ci interessa dove stanno gli elementi su righe e colonne, usiamo Flex quando ci interessa l'allineamento in una sola direzione.

2) Il grid container: display: grid e grid-template-columns

aggiungendo display: grid i figli diretti diventano grid-items.

3) Tracce e unità: px, fr, auto, minmax()

Una "traccia" di Grid è una colonna o una riga.
Possiamo usare varie unità per dire quanto è grande:
px, %, fr, auto, minmax().

Usiamo fr per dividere lo spazio in parti e minmax() + autofit per layout responsivi "che si aggiustano da soli".
